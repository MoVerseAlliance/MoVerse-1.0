
Verse 1.0

MoVerse is a decentralized professional services platform that connects employers with professionals (e.g., UI designers, video editors, CG artists, voice-over artists). This is the 1.0 version, focusing on core features like task posting, bidding, and task completion.

## Features
- Task posting (regular and urgent tasks with a 10% urgent fee).
- Anonymous bidding (price-lowest-wins mechanism).
- Task completion and winner selection.

## Contracts
- `MoToken.sol`: ERC-20 token for payments (MoVerse Token, MO).
- `Task.sol`: Manages tasks, bidding, and completion.

## Setup
1. Open Remix (https://remix.ethereum.org).
2. Upload `MoToken.sol` and `Task.sol`.
3. Compile and deploy using Remix VM (Cancun).
4. Test core features (task posting, bidding, completion).
